# 📚 docs

Repositório para tutoriais, dicas e anotações sobre diversos temas de tecnologia e desenvolvimento de software.

## Sobre este repositório

Este espaço é utilizado para organizar e compartilhar conhecimento acumulado ao longo do tempo, incluindo:

- **Dicas** — atalhos, boas práticas e truques úteis do dia a dia
- **Tutoriais** — guias passo a passo sobre ferramentas, linguagens e tecnologias
- **Anotações** — resumos e referências rápidas sobre temas variados
